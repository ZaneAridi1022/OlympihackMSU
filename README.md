## Getting Started


### Prerequisites
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ZaneAridi1022/TrustLink.git
   ```
2. Move to Client
   ```sh
   cd TrustLink & cd client2
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4.Start the client
 ```sh
   npm run dev
   ```
